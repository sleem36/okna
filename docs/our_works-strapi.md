# Страница «Наши работы» — данные из Strapi

Страница `/our_works` выводит разметку как в эталоне [`docs/our_works-page.html`](our_works-page.html). Контент заполняется через админку Strapi.

## Откуда берутся данные

- **Список работ (карточки в сетке):**  
  Single Type **Home Works Gallery** (`/admin/content-manager/single-types/home-works-gallery`).  
  Внутри — компонент с массивом **works** (компонент «Работа»).

- **Категории для табов:**  
  - Либо из **Portfolio Categories** (Collection Type), если заполнены — тогда табы «Все» + список категорий из Strapi.  
  - Либо используются встроенные: Все, Террасы, Беседки, Дачи, Веранды, Промышленные помещения.

- **Если в Home Works Gallery нет работ:**  
  На странице подставляются записи из **Portfolio Items** (коллекция `portfolio-item`).

## Поля одной работы (в Home Works Gallery → works)

| Поле         | Описание |
|-------------|----------|
| **title**   | Название под картинкой |
| **slug**    | URL карточки: ссылка «Посмотреть» ведёт на `/our_works/[slug]` |
| **categorySlug** | Таб: `terrasy`, `besedki`, `dachi`, `verandy`, `promyshlennye` (или пусто для «Все») |
| **image**   | Медиафайл из Strapi (если используете Media Library) |
| **imagePath** | Путь к картинке в проекте, например `/wp-uploads/2023/05/пер.jpg` (файл лежит в `frontend/public/wp-uploads/...`) |

Приоритет картинки: если задан **imagePath**, он используется; иначе — **image** из медиатеки.

## Фильтр (Окантовка, Основание монтажа, Материал)

Кнопки «Применить» и «Сбросить» работают. Чтобы фильтр реально отсекал работы, в компоненте работы (или в Portfolio Item) нужно завести поля, например:

- **edgingType** — окантовка: `standart`, `luxe`, `premium`
- **mountBase** — основание монтажа: `wood`, `metal`, `brick`
- **material** — материал: `pvh`, `poly`, `tone`

Пока этих полей нет в схеме — фильтр только собирает выбранные чекбоксы; при добавлении полей в Strapi фильтрация по ним заработает без смены страницы.

## Разметка страницы

Структура совпадает с эталоном:

- Секция с заголовком «Наши **работы**»
- Табы (Все, Террасы, …) — первый блок внутри контента
- Обёртка: слева форма фильтра, справа сетка карточек по табам
- Карточка: изображение, оверлей «Посмотреть» по наведению, подпись, ссылка на `/our_works/[slug]`

Данные для заголовка и работ редактируются в Strapi; после сохранения обновите страницу (или перезапустите сборку, если используется SSG).
